# HUSTOJLUOGU
让用户、教师、学生更轻松的爬取洛谷网站上的题目，并合成适合于HUSTOJ导入的XML文件，以及加入了__client_id的设置，使用户可爬取必须用户登录的题目、内部题目(T)、以及比赛题目。
请使用PYTHON-IDE(64)打开并运行，如开始运行遇到报错，请对照报错安装你所缺少的依赖。
请打开www.luogu.com.cn,打开浏览器开发人员模式，切换至"Network"选项卡，随意点击一个题目，切换至"COOKIES"选项卡，即可查看到__client_id，一大串字母类似于：ykqxbkv4l2olzrmb*******，这串字母是你用于登录洛谷的令牌，切勿泄露。（导入令牌后即可爬取内部题目）
祝你玩得愉快！
